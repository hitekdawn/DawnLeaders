# DawnLeaders

A World of Warcraft addon for the Dawn community's squad leaders

## Features

- Generate comma-separated lists of raid members with their realm names
- Filter by raid groups (1-8) with checkboxes
- Quick copy to clipboard functionality
- Keybind support for easy window toggle

## Usage

### Slash Commands

- `/dawnatd`
- `/gws`
- `/atd`

### Keybinds

You can set a keybind to toggle the attendance window in the game's Keybindings menu under "DawnLeaders".

### How It Works

1. Open the attendance window using any slash command or keybind
2. Select which raid groups (1-8) you want to include using the checkboxes
3. The list automatically updates as a comma-separated list with player names and realms
4. Press Ctrl+C to copy and close the window, or Escape to just close
